== Description ==

Consumer Withdrawal for WooCommerce makes it easy for online stores to comply with the European Union consumer withdrawal legislation.

Customers can submit a withdrawal request directly from their My Account page without contacting support manually. Store owners receive instant email notifications and can manage all requests from the WooCommerce administration area.

The plugin supports both complete and partial withdrawal requests, making it suitable for stores selling multiple products within the same order.

Designed specifically for WooCommerce, it integrates seamlessly into existing stores without affecting the checkout process.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen.
3. Make sure WooCommerce is installed and activated.
4. Configure the plugin under WooCommerce → Consumer Withdrawals.
5. Customers can now submit withdrawal requests from their account.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes.
WooCommerce must be installed and activated.

= Can customers withdraw only specific products? =

Yes.
Customers can submit withdrawal requests for selected products within an order.

= Does the plugin send emails automatically? =

Yes.
The administrator receives a notification email and the customer receives a confirmation email immediately after submitting a request.

= Can I manage requests from WooCommerce? =

Yes.
All withdrawal requests appear inside WooCommerce and include:

* Status
* Request date
* Selected products
* Customer information

= Is the plugin GDPR compliant? =

The plugin stores only the information necessary to process withdrawal requests.

= Is the plugin compatible with HPOS? =

Yes.
The plugin supports WooCommerce High-Performance Order Storage (HPOS).

= Can I translate the plugin? =

Yes.
All strings are translation-ready using the WordPress internationalization functions.
